# Prendre un instantané d'une VM sous Virtual Box

Pour faire un instantané :
 - Sélectionner la VM
 - Cliquer sur le bouton à droite du nom qui représente une liste d'éléments puis sur `Instantanés`
 - Cliquer sur le bouton `Prendre`
 - Entrer un nom, comme par exemple `Fresh install`. On peut entrer une description si on souhaite que l'instantané soit plus clair et compréhensible à l'avenir, si le nom n'est pas assez explicit
 - Cliquer sur `OK` une fois cela remplit
 - Cliquer sur le bouton à droite du nom qui représente une liste d'éléments puis sur `Détails`
 - Relancer la VM en cliquant sur le bouton `Démarrer`
